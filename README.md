# Cricket Scoreboard Simulator

## Description
A Cricket Scoreboard Simulator developed in C that simulates batting, bowling, player statistics, and match management.

## Features
- Player Management
- Batting Simulation
- Bowling Simulation
- Match Summary
- Score Calculation

## Technologies Used
- C Programming
- GCC Compiler

## How to Run

Compile:
gcc main.c batting.c bowling.c player.c match.c stack.c utils.c -o cricket

Run:
./cricket
